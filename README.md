# 🌈 ¡Feliz Cumpleaños para la más Matemática! 🦄

Este proyecto es una página web especial creada con todo el amor del mundo para celebrar el cumpleaños de la persona más increíble, ¡mi novia! Inspirado totalmente en el universo de **Hora de Aventura** (Adventure Time).

## ✨ Características Especiales

- **🏰 El Reino de Ooo en tu Navegador**: Un diseño vibrante y colorido que recrea la magia de la serie.
- **Finn & Jake**: No podían faltar los héroes principales flotando por la pantalla.
- **👑 Realeza y Más**: Incluye a la Princesa Bubblegum, Marceline, el Rey Helado, Mentita, Tronquitos y más.
- **🐧 Gunter & BMO**: Los compañeros más fieles están aquí para celebrar contigo.
- **✨ Detalles Mágicos**: Nubes animadas, estrellas brillantes, corazones y una lluvia de confetti interactiva.
- **💖 Mensajes con Amor**: Un diseño pensado para sacar una sonrisa en cada rincón.

## 🚀 Tecnologías Utilizadas

- **React**: El corazón de la aplicación.
- **Tailwind CSS**: Para un diseño moderno y responsivo.
- **Framer Motion**: Para las animaciones fluidas y mágicas de los personajes.
- **Lucide React**: Iconos hermosos para los detalles.

## 🛠️ Cómo Ejecutar el Proyecto

1. **Instalar dependencias**:
   ```bash
   npm install
   ```

2. **Iniciar el servidor de desarrollo**:
   ```bash
   npm run dev
   ```

3. **¡Disfrutar de la aventura!**
   Abre [http://localhost:5173](http://localhost:5173) en tu navegador.
